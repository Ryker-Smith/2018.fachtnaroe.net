Java Platform, Micro Edition Embedded (Java ME Embedded) 8.3

Supported APIs

Java ME Embedded platform is an implementation of JSR 360 and JSR 361 specifications, which also supports several optional packages and additional APIs. 



Reference Implementation APIs

  • (JSR 360) - CLDC 8
  • (JSR 360) - GCF 8
  • (JSR 361) - Java ME Embedded Profile 8

Optional Package APIs

  • (JSR 75) - PDA Optional Packages for the J2ME Platform
  • (JSR 120) - Wireless Messaging
  • (JSR 172) - Java ME Web Services
  • (JSR 177) - Security and Trust Services API
  • (JSR 179) - Location
  • (JSR 280) - XML API

Oracle Java ME Embedded APIs

  • System Configuration API
  • DIO 1.1
  • OAuth 2 API
  • JSON API
  • HTTP Client API
  • Security API
  • Certificate and Key Storage Control API
  • Runtime Update API
  • SSL Extension API
  • SATSA Extension API
