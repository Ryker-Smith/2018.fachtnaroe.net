Assessment 2007/8

Assessment is by three elements:

  • Examination (Theory-Based): 40%
    • The exam will be posted afterwards
  • Assignment: 30%
    • Analyse the Senior College PLC Application Process (PDF) (HTML format)
    • FAQ: Common questions & answers relating to the project
  • Skills Demonstration: 30%
    1. Locate, print and save the (out-of-date) PDF PLC application form for the Senior College on the official CTI website.
    2. Create an entry for Roe, Fachtna in your address book. The email address used should be that given to you previously. Create one other entry in the address book of a friend or family member. Capture the screen showing your address book; print and save the image.
    3. Create an alternate application form (a DCF for PLC applicants) for the college. It need not cover all possible courses or be completely accurate in all content. i.e. Fully correct phone numbers, addresses, email addresses and course sponsorship details would be optimal - but are not necessary. Apart from the Open File Format version of the file, generate a PDF version also.
    4. Create a new email; address it to Roe, Fachtna, the other new entrant in the address book and to info[at]informationtechnologyservices.ie - attach to it the previous version of the CTI PLC application form, your new version of the form and the captured image of your address book; use the subject line ICS Submission and send it with a suitable cover note describing both the attachments and the purpose of the email.
