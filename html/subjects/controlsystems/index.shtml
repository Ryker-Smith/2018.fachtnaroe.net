Introduction to Robotics & Embedded Computing (AKA Control Systems)

This module is designed so that the learner will understand the vital role that microprocessor control plays in modern industrial processes.

The module is designed so that the learner will learn how to use a microprocessor based device as a controller for physical devices.

With this knowledge you should be able to design and build a control system, needing a number of inputs and outputs, combined with feedback to solve a practical problem.
