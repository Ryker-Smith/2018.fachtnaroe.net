The exam instruction
Task 2         (8 marks)
Design and create a data input screen (FORM) and then use
the data input screen to enter the sample`data shown in
Figure 2 into the database structure ‘CASUALTY’ created
in task 1.
With regard to the data input screen you should follow these
guidelines:
a) All fields must have suitable field labels.
b) The input screen should have a title on top and centred.
c) There is no need for borders or other effects.
Your invigilator will award marks to your screen design
during the examination.
