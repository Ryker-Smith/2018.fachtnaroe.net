Lotto Numbers Program
Write a program that generates sets of 6 lotto numbers and displays them on-screen.

Overview of Program Flow
When the program is run the user should be asked how much money they wish to spend on the Lotto. The user will type the amount of money (in pounds) and the program should generate the appropriate number of sets of 6 numbers (6 numbers for 75p).

When the program has displayed the sets of numbers on the screen the user should then be asked if they wish to generate more sets of numbers. If the user declines then the program finishes. (This test may alternatively be carried out prior to generating any numbers).

Restrictions
1. Negative monetary values are not accepted and must be handled in some way.
2. A minimum of two plays must be enforced. (A minimum play is £1.50.)
3. Additional plays are £0.75 each.
4. Numbers generated must be in the range of 1-42.

Assumptions
1. No allowance for duplicate numbers need be made.

Test Data
 
Input Data Expected Result
-3.00  Error message generated
1.50 2 sets of 6 numbers
3.00 4 sets of 6 numbers
0  No numbers generated
3.75 5 sets of 6 numbers

Submission Criteria
Submission must include:
1. Pascal source code,
2. Flow chart created in SmartDraw

<comprog02@cti-clonmel.ie>

Write a program which generates sets of 6 random Lotto numbers (Range 1-42) and prints them onto the screen for the user.

When the program is run the user should be asked how much money they wish to spend on the Lotto. The user will type the amount of money (in pounds) and the program should generate the appropriate number of sets of 6 numbers (6 numbers for 50p).

When the program has displayed the sets of numbers on the screen the user should then be asked if they wish to generate more sets of numbers. If the user declines then the program finishes.
The program will include: 3 WHILE loops
The RANDOM function

NOTE: You do not need to worry about duplicate random numbers being selected.
