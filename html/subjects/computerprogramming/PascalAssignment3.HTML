Academic Year 1999/2000

Computer Programming (C20013)
Assignment 3
January 2000

Brief: Write a computer program with a bilingual option. The programs converts marks to grades.

Program Function: The program will read in a series of exam results and out put the grades which those marks represent. The program should use a simple ARRAY to store the data, as well as WHILE loops and IF statements where necessary.

Program Limitation: The user should be asked at the start which language they wish to use for the progam. If they select French, then all input and output should be in French. If they choose English then all input and output should be in English.

Program Sample Run (English): 

Program Starts.

Enter ten marks, please.
Mark ( 1): 23
Mark ( 2): 54
Mark ( 3): 83
Mark ( 4): 68
Mark ( 5): 12
Mark ( 6): 97
Mark ( 7): 150
Mark ( 8): -23
Mark ( 9): 62
Mark (10): 57

NCVA Computer Programming, Level 2, Module C20013

================ASSIGNMENT #3=================

Write a program to allow the user to enter 10 students exam marks into an ARRAY.

After all of the data is entered it should then be written out on screen with the corresponding grade for that mark.

The program should use a simple ARRAY to store the data, as well as FOR loops and IF statements.
