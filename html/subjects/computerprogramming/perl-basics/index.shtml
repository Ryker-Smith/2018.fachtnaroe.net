Perl Basics - Getting Started

A first program

Open an SSH session on the programming server. When connected create and edit a new file called hello.pl using the vi editor by typing the command:

vi hello.pl

As soon as the editor starts press i to go to INSERT mode. If you do this correctly the word --INSERT-- will appear at the bottom left of the screen. Type in this program...

#!/usr/bin/perl
print "Hello World!\n";

...exactly as shown. The file will initially not be executable so to make it so, type:

chmod +x hello.pl

You should now be able to run the program by typing:

./hello.pl

The output of this program should look like this...

[username@mymailbox ~]$ vi hello.pl
[username@mymailbox ~]$ chmod +x hello.pl
[username@mymailbox ~]$ ./hello.pl
Hello World!
[username@mymailbox ~]$

...if all goes well. If anything fails to work as expected, go back through the steps again checking thoroughly to find your mistake(s).
