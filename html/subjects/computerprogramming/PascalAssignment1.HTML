Currency Converter Program

Write a program to convert Euro values to Punt values & vice-versa.

Restrictions

1. Negative values are not accepted and must cause the program to halt.
2. The amount to be converted must be accepted before either sourceor destination currencies are accepted.
3. Final answer should be displayed correct to two decimal places.

Assumptions

1. The Euro to Punt conversion factor is 0.787564
2. The letters e/E and p/P can be used to represent Euros andPunts, respectively.

Test Data

Input Data Expected Result
-17 Program should halt
25 E P19.69P
25 P E31.74E
0  P E0.00E
25.578 E E25.58E
25 P P25.00P

Submission Criteria

Submission must include:
1. Pascal source code,
2. Flow chart created in SmartDraw

Submission materials should be e-mailed as attachments to
<comprog01@cti-clonmel.ie>
as well as submitted in paper form to the submission box in Room 15.

================ASSIGNMENT #1================


Write a computer program that READs four numbers.

These numbers represent the lengths of the four sides of a shape. Yourprogram must determine IF the shape could possibly be a square.

If the shape could be a square then your program will WRITE this facton screen. If the shape could not possibly be a square then your programwill WRITE this fact on screen.

IF any of the numbers entered are negative then the program should WRITEthat invalid data has been entered and stop immediately without performingany other tests on the data.

You must use the PASCAL programming language to complete the assignment.
