Hidden input fields demo 02

This demo shows how data from input fields can be incorporated as hidden in a later form.

This is the first input form:

<!DOCTYPE html>
<html>
<head>
<title>Hidden input demo #02</title>
</head>
<body>
<h2>Hidden input fields</h2>
<p>Enter two items (text or numbers) in the boxes</p>
<form action="cgi-bin/hidden_input_02.pl" method="post">
<input type="text" name="field_1">
<input type="text" name="field_2">
<input type="submit" value="Press to send these">
</form>
</body>
</html>

This is the perl program:

#!/usr/bin/perl -T
use strict;
use CGI::Carp qw(fatalsToBrowser);
use CGI;

our (%in); # This is the variable used by CGI::ReadParse below
CGI::ReadParse(); # Tell the module to 'parse' the form 
my ($field1, $field2, @filecontent);
$field1 = $in{field_1};
$field2 = $in{field_2};
&content_type;
open FILE1, "< ../hidden_input_02a.html";
open FILE2, "< ../hidden_input_02b.html";
@filecontent = <FILE1>;
print @filecontent;
print "<input type=\"hidden\" name=\"field_1\" value=\"$field1\">";
print "<input type=\"hidden\" name=\"field_2\" value=\"$field2\">";
@filecontent = <FILE2>;
print @filecontent;
close FILE1;
close FILE2;

sub content_type {
# standard html content-type string
  print "Content-Type: text/html\n\n";
}
