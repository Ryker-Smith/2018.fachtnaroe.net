Site organisation on disk

  • Site should be well organised on disk

  • Use of directories for sections is normal

    • also for images, scripts

  • Avoid putting all files in one directory

    • makes updates harder but improves order

  • References to files in other directories should specify path to those files

  • Avoid use of spaces in file and directory names

    • use only lowercase - to match the links!
