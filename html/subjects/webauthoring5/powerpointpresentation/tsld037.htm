The change-reload cycle

  • When designing a page you’ll need to use both the composer and the browser to operate

  • Create the page and save it

  • Alternate to the browser and load the page

  • Return to the composer and make and save your changes

  • Return to the browser and click ‘reload’ to see the changes

  • For greater efficiency, this entire operation may be keyed
