WHY  ACCESS ?
In search for a development application to suit our needs the primary scope of suitable choices were Visual Basic, Paradox and Access.

 Microsoft Access is part of the Microsoft Office, a family of full-featured products designed to work together like one.  Access is a flexible and powerful relational database manager for Microsoft Windows and therefore the name Access is an accurate description for this program.  Access stands out in the computer world as having the ability to use more different types of data, in there original forms, than any other product currently on the market.

 Access is ‘user friendly’ in the way that it allows the programmer to take full advantage of the Windows environment.  It provides an excellent menu-driven interface and quick and easy to use tool bars.  Access provides an excellent on line Help with the option of also using a valuable guide in the form of Cue Cards.  It allows you to take advantage of Access Basic , a powerful programming language built into the product, which allows you to develop complete applications with the run time version of Access.
