Web Site Development - One possible method

Instead of trying to develop all pages simultaneously, first focus on one page that has all the style and layout elements that you will later use on all pages. First create one empty file containing just the basics:

<!DOCTYPE html>
<html>
<head>
<title>
</title>
</head>
<body>
<head>
<div id="banner">
</div> <!-- #banner -->
<div id="menu">
</div> <!-- #menu -->
<div id="content">
</div> <!-- #content -->
</body>
<html>
<>

Save this file as test.shtml in your web path.

Use a Lorem Ipsum generator to get some text as a greeking equivalent. Just enough text to comfortably fill a screen with a little over should do. Paste the Lorem Ipsum text into your file between the start and end of the content div.
