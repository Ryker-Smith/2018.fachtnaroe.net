xbox controller sample

This code sample shows how the xboxdrv program can be used to enable the xbox controller to run with the previous code sample.

This is the content of a bash executable, which calls and configures the xboxdrv program, before starting the game program. In this case, the fire buttons are 'space' and 'k'. The movements left/right and up/down are mapped to the axes of the two joysticks on the controller.

#!/bin/bash
sudo xboxdrv --type xbox --ui-axismap x1=KEY_LEFT:KEY_RIGHT,y2=KEY_UP:KEY_DOWN --ui-buttonmap guide=KEY_X,RT=KEY_SPACE,LT=KEY_K --detach-kernel-driver ./wingsOfwarMultiShot.pl
