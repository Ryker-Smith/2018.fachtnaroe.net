Audio

  • Audio input is primarily capture only

  • Audio interpretation has long been possible though unreliable

  • System control as well as data input are possible

  • Accuracy depends primarily on the algorithms used to recognise speech

  • The processing power of the system should be as high as possible – including sufficient amounts of RAM

  • All systems currently need to be trained to recognise the intended user
