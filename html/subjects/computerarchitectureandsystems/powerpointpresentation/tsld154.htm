Decimal To Hexadecimal Example #1

    Sample: 2610

    Place: 4 3 2 1

    Value: 163 162 161 160

    Value: 4096 256 16 1

    RULE: Divide by value of each place in turn, if possible. If possible, write answer using digits 1..F Write 0 if not possible. Continue with remainder.

    Result: 0 A 3 2
