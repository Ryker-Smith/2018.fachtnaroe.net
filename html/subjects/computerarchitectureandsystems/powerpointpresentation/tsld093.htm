Serial Transmission Control

  • Flow control may be carried out in hardware or software

    • CTS/CTRTS is hardware control
    • XON/XOFF flow is software control

  • Hardware flow control is generally better

  • ASCII codes used for software flow control

  • Standard serial ports controlled by 16550 Universal Asynchronous Receiver Transmitter (UART)
