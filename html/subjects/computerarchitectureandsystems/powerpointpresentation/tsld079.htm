Keyboard

  • Older PC Keyboards have 89, 101 or 102 keys

  • Modern Windows keyboard has 105 keys, including dedicated Start button

  • May connect via PS/2 USB or DIN-5 port

  • Cost c£20

  • Number pad is non-essential

    • Early keyboards didn’t have them
    • UNIX sysop keyboards don’t have them

  • PC-XT keyboards had function keys at side
