How RAM Is Added

  • Single In-Line Memory Modules (SIMM)

  • Dual In-Line Memory Modules (DIMM)

  • DIMMs used in Pentium 2 and 3

  • SIMMs used in other Pentiums - must be added in pairs

  • Finite number of sockets

  • 64/128MB suitable for modern multimedia systems
