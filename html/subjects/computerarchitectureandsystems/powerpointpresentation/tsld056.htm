DMA Data Transfer

  • DMA (Direct Memory Access) Specialised circuitry or a dedicated microprocessor that transfers data from memory to memory without using the CPU.

  • PCs have eight DMA channels. Many peripherals use DMA, e.g. sound cards

  • Although DMA may periodically steal cycles from the CPU, data is transferred much faster than using the CPU for every byte of transfer.
