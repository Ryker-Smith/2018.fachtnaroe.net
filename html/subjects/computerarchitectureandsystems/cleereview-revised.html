This document should be read in conjunction with the original project brief, which describes the CleereView Communications company.

 

Provide the following:

 

Minimum, one standard formatted A4 page containing, at a minimum, one paragraph on each of:

  • Brief overview, in your own words, of the CleereView company.
  • Brief overview of their existing technology.
  • Brief overview of the suggested technological solution.
  • Brief description of the expected benefits of the new solution.

 

As many standard formatted pages as are necessary to describe:

  • Description of any one workstation which will be provided for an employee based in the office. This will be considered the office standard.
  • Description of any one workstation which will be provided for an employee based in the field. This will be considered the field standard.
  • Description of any one workstation which will be provided for an employee alternating between the office and the field. This will be considered the mobile standard.

The above computers can be of any type, PC, Mac, UNIX, laptop, desktop as required. Description is of hardware only.

 

As many standard formatted pages as are necessary to describe:

·         Hardware description of the server.

·         Software description of the server, including any main functions, such as file-server etc.

·         The networking solution to be installed. Specify speed of network, hubs etc. Include any dial-up items here.

 

Maximum, one standard formatted A4 page describing:

·         Office (business) software to be used. This will be considered the standard business software bundle.

·         Artistic/creative software to be used. This will be considered the standard creative software bundle.

·         Communications (email/web) software to be used. This will be considered the standard communications software bundle.

 

As many standard formatted pages as are necessary to describe:

·         All peripherals which you feel should be provided.

 

A chart of all staff (use numbers, or imaginary names if you prefer) showing:

·         Category (Figure 1 in original document) of staff member.

·         Location of work place.

·         Standard computer allocated (from above).

·         Software bundles allocated (from above).

·         Specific peripherals, if appropriate.

·         Mode of connection to the company network.

 

A chart of all costs, which need only be estimates, and upon which there is no upper limit showing:

·         Total number of workstations and estimated cost

·         Servers and estimated cost

·         Peripherals and estimated cost

·         Workstation software bundles and estimated cost

·         Server software and estimated costs

·         Any other costs

·         Total of all cost.

 

Maximum, one standard formatted A4 page giving:

·         A description of how you expect the system to be used.

·         Conclusions of your project, with any final recommendations.

·        Sources of your data (bibliography)
