Exit Windows
Do you need to automatically shut down, reboot, or log out of a Windows PC? This may be what you need. 

I have Windows 95 PC that runs constantly. Well, it would if the OS could! I use a scheduling program, so I thought I'd have it reboot my system every weeknight at 3am. At first I was using the rundll program to activate the Windows shutdown functions. But I had a dickens of a time using the 32-bit methods or passing arguments. This meant I couldn't "force" the issue. There is only so much you can do with a command line. One stupid program with an error message on the screen would hang the entire machine. 

So I was forced to write my own program. It has no problem doing exactly what you want. You can log off, reboot, or fully shut down your PC. And you can force it. The included "readme" file tells you everything you need. Heck, there are only four optional command-line arguments, and if you forget them or lose the readme, you can just check the file properties to get a reminder. 

To get the program, just click the "Download Now!" at the top of the screen. Full VB5 source code is included. This program needs the Visual Basic 5 runtime files which you probably already have. So I didn't include them in the download! If you need them, you can get from Microsoft or Ziff-Davis.
