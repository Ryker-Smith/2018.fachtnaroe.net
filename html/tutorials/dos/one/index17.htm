Pong for Microsoft Excel 97
 
 
This is a VBA module that can be dropped into any Excel97 spreadsheet to give it single-player Pong-playing capability.  This was done just because it can be done. What other reason is there? 

As it is currently written, pressing the "F12" key starts Pong, and pressing the "Escape" key stops Pong. The right and left arrow keys move your paddle. If you hit the ball with the side of the paddle, you change the angle of the resulting bounce. You get a "beep" if you miss the ball. I usually miss, so I hear lots of beeps. Because I miss so much, I decided Pong wouldn't keep score. The longest recorded time before becoming bored with the game is 45 seconds. Pong doesn't affect any data on your spreadsheet.

This is actually a demo of how to use callbacks from inside VBA. You aren't actually supposed to be able to do it! VBA doesn't support the "AddressOf" function, and without it you can't find the address of a function for a callback. Of course, there is always a low-level undocumented workaround! Here I use a Windows timer API function and convince it to call back to one of my routines so I can move the pong ball every few milliseconds.
