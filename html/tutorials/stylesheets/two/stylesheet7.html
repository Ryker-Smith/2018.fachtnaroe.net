This tutorial was by no means a complete guide to stylesheets. There are many other properties that I choose not to include for reasons of simplicity or for the fact that their support in browsers was very weak. What I included was all the properties that you need to know in order to produce pages that go beyond the normal design ideals.

I have included two example pages that show what can be done using stylesheets. As we discussed before, the support for stylesheets can be a bit shaky, so to properly view these pages you should have IE 4.0+ on a Windows platform.
