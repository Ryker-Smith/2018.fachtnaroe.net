-Conclusion 		LESSON THREE: Controlling Layout

Now that we’ve gone over how to change the size, color, and face of your font, we’re going to delve into how to gain control over the spacing, indenting and other layout properties of your text.

LETTER SPACING

This property allows you to either increase or decrease the space between letters on the screen.

p { letter-spacing: 3 px }
h2 { letter-spacing: -2 px }

Feel free to use any measurement unit that we discussed before when using this property.

LINE HEIGHT

This is one of my favorite things about CSS, because you now have full control over the space between lines of text which can be very useful if you have a lot of text and want to add spacing to increase the readability.

p { line-height: 12 pt }

Line height can also be specified using a number like “2” which is the same thing as double spacing. The browser will look at the size of the font you are using and determine what would be the equivalent spacing. You can also specify the line height with a non-integer number such as 1.3 or 1.5. Percentage values (ex: p { line-height: 150% }) are also accepted.

Although it may be tempting to use line-height to make text overlap, it is not the best way. Later on we’ll talk about the best way to accomplish this effect.

TEXT ALIGN

This one is pretty self explanatory.

h4 { text-align: right }

Your options are either left, right, center, or justify.

TEXT INDENT

Just use any of the measurement units (including percent) to specify the amount of indent. When you apply this to the <p> tag it will ONLY indent the first line.

p { text-indent: 5 px }
MARGINS

The four margin properties are:

margin-top
margin-bottom
margin-left
margin-right

You can apply all or just one of these properties, making sure that you separate multiple properties of the same tag with semi-colons.

p { margin-top: 20 px; margin-bottom: 15 }

Note that the word “padding” can be substituted for “margin” in the above examples with the same end result.
