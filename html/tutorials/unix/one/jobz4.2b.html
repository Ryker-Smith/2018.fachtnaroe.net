Understanding process information

The information displayed by the ps command varies acccording to which command option(s) you use and the type of UNIX that you are using.

These are some of the column headings displayed by the different versions of this command.

   UID\USER   PID   PPID   SZ   TT(Y)   TIME   COMMAND

Select a heading for a description of its meaning and content.
