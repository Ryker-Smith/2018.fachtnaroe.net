Useful ftp commands

When you come to use ftp on a regular basis you will find the following commands useful. They allow you more control over the file transfer process.

  • Making a directory on the remote host
  • Changing your local directory
  • Using Unix commands on your local host
  • Listing directories on the remote host
  • Changing directories on the remote host
