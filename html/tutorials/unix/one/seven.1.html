Move the cursor along a line

Press the ESC key to enter command mode before using these commands.

To move to                      Do this ...


next character                  l

previous character              h

next word                       w
next n words                    wn

previous word                   b
previous n words                bn

end of current word		e

start of current line		0 (zero)

end of current line		$
