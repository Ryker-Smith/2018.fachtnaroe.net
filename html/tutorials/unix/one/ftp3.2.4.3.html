Cancelling a file transfer request

You can cancel a file transfer request by removing it from the file transfer queue with the hhrm command.

To remove a transfer from the queue:

   hhrm transfer_id

To remove several file transfers from the file transfer queue simultaneously:

   hhrm transfer_id transfer_id transfer_id

Notice the space between each transfer_id.
