Transferring files

You can transfer files between hosts using the ftp command. This program is the standard means of transferring files between hosts on the Internet network.

Your system might also still support the use of the hhcp (host to host file transfer) command. This was supplied as part of the Colour Book Software (Blue Book) which used to be a standard within the UK Academic Community. See this note.
