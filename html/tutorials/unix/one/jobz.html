Managing jobs and processes

When you enter a command it invokes a program. While this program is running it is called a process. Although there is only one copy of a program, any number of processes can be invoked which run this program.

There are several commands which allow you to manage the processes that belong to you. You cannot do anything with processes belonging to other users.
