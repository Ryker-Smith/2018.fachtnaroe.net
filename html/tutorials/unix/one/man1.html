Hypertext online manual page services

There are two ways to use this service. You can either do a direct manual page lookup if the manual page name is known, or a keyword search of the whatis descriptions if you only have a rough idea of the particular manual page you are looking for.
  • man - direct manual page lookup
  • apropos - keyword search
Both services provide links to one another and all manual page references contained within the displayed manuals or whatis descriptions are replaced by appropriate links.
