umask: Tip

To make changes to the default access permissions which last until you next change them, place the umask command in your login file.

Which file this is depends upon the shell that you use:

Shell               login file


Bourne shell	    .profile
Korn shell	    .profile
BASH		    .profile
C shell		    .login
TC shell	    .login
