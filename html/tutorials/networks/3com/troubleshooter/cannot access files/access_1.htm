The Network Neighborhood Icon is Absent From the Desktop Screen:

The Network Neighborhood icon looks like this (unless you have personalized your icons):

If the Network Neighbourhood Icon is missing, check:

  • That Card Services are enabled on the laptop
  • The NIC or PCMCIA is correctly inserted
  • The NIC or PCMCIA card is correctly installed
  1. To check that Card Services are enabled on a laptop follow the instructions in Installing the PCMCIA card.

  2. To check the NIC or PCMCIA card is inserted correctly:

    Remove the NIC or PCMCIA card and re-install it. You may need to apply more pressure to ensure that the NIC is correctly seated in the slot. When the NIC is correctly seated, the gold connector on the edge of the NIC is normally completely inside the slot.
  • For instructions on inserting the NIC refer to its user guide and the section Adding a PC.
  • For instructions on how to insert the PCMCIA card you may need to refer to the user guides that accompany your laptop and PCMCIA card, and section Adding a Laptop.

  1. To see whether the NIC or PCMCIA card is correctly installed, you need to check that the card and drivers are correctly shown in the Device Manager. To check this, carry out the following instructions:
  1. From the Start menu, choose Settings and then Control Panel.

  2. In the Control Panel window, double-click System.

  3. In the System Properties dialog box, select the Device Manager tab.

  4. If the NIC or PCMCIA card is physically installed there will be a Network adapters entry in the listbox.
  • If Network adapters is not listed, the NIC and PCMCIA card is either incorrectly installed or may be faulty. If the NIC or PCMCIA card is faulty refer to its user guide for warranty and service details.
  • If the Network adapters entry is listed Double-click on the Network adapters item. It should expand to show the name of the NIC or PCMCIA card that has been installed. Go to step 4.
  1. If the NIC or PCMCIA card icon has a yellow circle and exclamation mark ('!') on it, it could due to the following:
  • There is a hardware conflict with another installed device, for example, a sound card.
  • The software drivers are either installed incorrectly or are the wrong type for the installed NIC or PCMCIA card. You need to change the drivers.

    To check whether there is a hardware conflict, carry out the following steps:

    1. Select the NIC or PCMCIA card in the listbox.
    2. Click on Properties.
    3. Select the Resources tab.

      If there is a conflict, a conflict message is displayed in the bottom listbox.
      If there is a device conflict use the Windows 95 or Windows 98 Hardware Conflict Troubleshooter.

To check your drivers configuration see Changing the Driver.
