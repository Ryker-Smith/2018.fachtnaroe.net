Device is Present in Network Neighborhood, but Cannot Open the File:

  1. Make sure that the PC or laptop you are trying to access is set up for File Sharing:
    1. From the Start menu, choose Settings and then Control Panel.
    2. In the Control Panel window, double-click Network.
    3. In the Configuration tab, click File and Print Sharing.
    4. In the File and Print Sharing dialog box, select both of the checkboxes.
    5. Click OK.
    6. Restart the PC or laptop.

  1. Make sure that the file has been set up for sharing and is not password protected. Refer to the section Using Your Network, for instructions on how to set up file sharing.
