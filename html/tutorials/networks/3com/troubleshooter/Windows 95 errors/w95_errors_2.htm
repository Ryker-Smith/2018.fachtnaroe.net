Unable to Browse the Network:

There are three main reasons why you cannot browse the network:

  • You have not logged onto the Network.
  • Your PC does not have the same Workgroup name as the other PCs on the network.
  • The Microsoft Browse Master has not recognized the new device.

To check that you have logged onto the Network:

  1. Restart the PC or Laptop.

  2. At the network login prompt enter your network user name and password.

  3. Click OK.

  4. Check the Network Neighborhood again. If you still cannot browse the network continue to the next page.

To check that your PC has the same Workgroup name as the other PCs on the network:

  1. From the Start menu, choose Settings and then Control Panel.

  2. In the Control Panel window, double-click Network.

  3. In the Network dialog box, select the Identification tab.

  4. The workgroup name for the PC is shown in the Workgroup field. Make sure the workgroup name (which is the name of the network) is the same for all of the PCs and Laptops that need to communicate with each other.

Microsoft Browse Master work-around:

When a new device is added to the network, it may take the Microsoft Browse Master some time to recognise that the new device has been added.

Instead of waiting for your PC's network information to be updated, you can use the Find option to find a specific PC or all PCs on your network:

  1. Click on the Start Menu.

  2. Click on the Find option.

  3. Click on the Computer option.

  4. In the Named field enter the name of the computer you are trying to find. The results of the search are shown in the Find listbox.
