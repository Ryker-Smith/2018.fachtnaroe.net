Your Network Adapter <Name> is Not Working Properly:

You have installed the NIC or PCMCIA card software drivers, but the NIC or PCMCIA card is incorrectly installed in the slot.

Remove the NIC or PCMCIA card and re-install it.

You may need to apply more pressure to ensure that the NIC is correctly seated in the slot. When the NIC is correctly seated, the gold connector on the edge of the NIC is normally completely inside the slot.

  • For instructions on Installing the NIC, see Adding a PC.
  • For instructions on how to install the PCMCIA card, see Adding a Laptop.

If this does not solve the problem see Network Troubleshooter in Windows Help.
