The Add New Hardware Wizard did Not Detect the NIC:

The NIC is probably incorrectly seated in the slot or may be faulty.
To get Add New Hardware wizard to detect the NIC you need to carry out the following instructions:

  1. Remove the NIC and re-install it. You may need to apply more pressure to ensure that the NIC is correctly seated in the slot. When the NIC is correctly seated, the gold connector on the edge of the NIC is normally completely inside the slot.

    See instructions on Installing the NIC in Setting up your PC. If re-installing the NIC does not solve the problem go to step 2.

  2. If you have a NIC that you know works, install this NIC in the PC. If it installs correctly, then the first NIC is probably faulty. Refer to your supplier for more details on warranty and service.

    If the NIC does not install correctly, there may be some problem with the configuration of your PC. The problem may be complex. Refer to your user guide and use any diagnostic tools available to you.
