The Configuration Screens are Not Shown when a PCMCIA Card is Installed:

  1. For a successful installation Card Services must be enabled on the laptop. To check this:
    1. Power on the laptop and start Windows 95 or Windows 98.

    2. From the Start menu choose Settings and then Control Panel.


    3. In the Control Panel window double-click PC Card (PCMCIA).
      • If the PC Card (PCMCIA) Properties dialog box appears (as shown below), Card Services is enabled. Go to step 3.

      • If the PC Card (PCMCIA) Wizard appears Card Services is not enabled. Go to step 2.

       

  1. To enable Card Services:
    1. In the Wizard select No and click Next>.
    2. Select No and click Next>.
    3. Click Finish.
    4. Restart the laptop and Windows 95 or Windows 98 as prompted.

If this does not solve the problem go to step 3.

  1. The PCMCIA card may not be correctly seated in the slot. Remove the PCMCIA card and re-install it. See the instructions in the Installing the PCMCIA card section. If this does not solve the problem, go to step 4.

  2. Try manually configuring the PCMCIA card. This is a similar process to manually configuring a NIC, as described in Configuring the NIC. If this fails go to step 5.
  3. Try installing a PCMCIA card that you know works.
    1. If it installs correctly, then the first PCMCIA card is probably faulty. Refer to your supplier for more details on warranty and service.
    2. If the working PCMCIA card does not install correctly the problem may be more complex. Refer to your user guide and use any diagnostic tools provided.
