Network Appears Slow When Users Try to Access a Particular PC/Printer:

If a lot of users are sharing resources on the same PC, that PC's response time will be very slow. This is because the PC has to deal with a lot of different requests, and each request uses some of the PC's processing power. For example:

  • The owner of the PC is using the PC to perform day-to-day tasks, such as opening, editing and saving files.
  • Other users may also be opening, editing and saving shared files on that PC.
  • If a printer is attached to the PC, the PC is also having to process printer jobs.

To resolve the problem you can:

  • Move the Printer to another PC, and only use that PC to process print jobs.
  • Add another printer to reduce the demand on the original printer.
  • Put the shared files on a different PC, or share the files between several PCs.
