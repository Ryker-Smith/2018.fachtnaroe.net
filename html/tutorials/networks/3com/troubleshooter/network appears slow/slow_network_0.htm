Network Appears Slow:

The ability of your network to transfer information is measured in terms of available bandwidth.
It is normal for networks to experience high bandwidth usage from time to time. If someone is transferring a large file across the network, the file may take up most of the available bandwidth and make the network appear slow to other users.
There are certain situations when a slow network is unacceptable. You should consider upgrading your network if the network always appears slow or there are users that must have guaranteed bandwidth available at all times.
