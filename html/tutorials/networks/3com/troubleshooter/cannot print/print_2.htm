You Can See the PC That Has the Attached Printer in the Network Neighborhood, but Still Cannot Print:

Check that:

  • The printer is switched on.
  • The printer is on-line. If the printer is not off-line, or the printer is giving a different error message, refer to the user guide for instructions.
  • There is paper in the paper tray and that the printer is set to use that tray.
  • The toner or ink cartridge does not need replacing.
