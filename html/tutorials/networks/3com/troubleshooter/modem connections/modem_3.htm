The Internet Connection Wizard is Displayed Instead of the LAN Modem Configuration Screens:

When you click on your Web browser icon you may see the Internet Connection Wizard screen instead of the LAN Modem configuration screen you were expecting. This happens when you try to access the Internet for the first time.

Note: The example shown below is based on Windows 95. Your screens may be slightly different for Windows 98, but the principle is the same. You must tell your computer that the LAN Modem should be contacted over the network rather than through a serial connection.

To ensure that the PC can communicate with the LAN Modem complete the following steps:

  1. Click Next. The Setup Options screen is displayed.



  2. Make sure that you select the middle option (as shown in the screen above).

  3. Click Next. The Set Up Your Internet Connection screen is displayed.



  4. Select Connect using my local area network [LAN] option.

  5. Click Next. The Proxy Server screen is shown.



  6. Select the No option.

  7. Click Next. The Setup Your Internet Mail Account screen is shown.



  8. Select the No option.

  9. Click Next. The Set Up Your Internet News Account screen is displayed.



  10. Select the No option.

  11. Click Next. The Set Up Your Internet Directory Service screen is shown.



  12. Select the No option. The Complete Configuration screen is shown.



  13. Click Finish.

  14. Configure your LAN Modem as described in Setting up the WebWizard for WAN access.
