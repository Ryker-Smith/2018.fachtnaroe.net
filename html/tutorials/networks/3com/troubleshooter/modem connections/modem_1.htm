Initial Communication Between the LAN Modem and an Attached Computer Cannot Be Established:

This section helps you troubleshoot an OfficeConnect LAN Modem. If you have a different type of modem, refer to the troubleshooting section in your modem's user guide.

If you have connected your OfficeConnect LAN Modem to a PC or laptop that has been set up to automatically obtain an IP address and the PC or laptop cannot communicate with the Modem, follow these instructions:

  1. Check that the Power LED on the OfficeConnect LAN Modem is lit. If it is not lit:
    1. Check that the power cable is connected and that the OfficeConnect LAN Modem is powered on at the wall socket.

    2. If the Power LED is still off, check that you are using the correct power supply. Use the power supply that accompanied the modem.

    3. If this does not solve the problem and you have another OfficeConnect LAN Modem power supply that you know to be working, use that power supply. If the Power LED comes on, the original power supply was faulty. Check your user guide for warranty and support information.

    4. If you are using a power supply that you know works and the Power LED is still off there may be a problem with the OfficeConnect LAN Modem. Check your user guide for warranty and support information.

     

  1. Check that the LAN Port Status LED is lit. If it is not lit:
    1. Check that one end of the Ethernet cable is securely connected to a port labeled either 1, 2, 3 or 4 on the modem, and that the other end is securely connected to the NIC or PCMCIA card port on the PC or laptop.

    2. If this does not solve the problem, check that you are using the correct cable. Use the cable labeled "Ethernet" that accompanied the modem. Do not use a telephone cable to connect the PC or laptop to the modem. If possible use an "Ethernet" cable that you know to be working. If this does not solve the problem go to the next step.

     

  1. If the NIC or PCMCIA card has only just been installed in your PC or laptop, check that it is correctly seated in the slot and correctly configured. If necessary, reinstall the NIC as described in Adding a PC or PCMCIA card as described in Adding a Laptop.

  2. Check that TCP/IP is correctly installed with its default values, as described in Setting up TCP/IP.

  3. If this does not solve the problem you may need to reset the IP Address on your PC or Laptop:
    1. Find and run the file c:/windows/winipcfg.exe

    2. Click on the "Release All" button.

    3. Click on the "Renew All" button.

     

  1. If the PC or laptop still cannot communicate with the modem, enter the IP address of the LAN Modem in your Web browser. You can enter either:

    http://192.168.1.1/mainpage
    or
    http://3com.OC.lanmodem/mainpage.

    If the PC or laptop can communicate with the modem, the modem configuration screens are shown. If the configuration screens are not shown go to the next step.

  2. Reset the LAN Modem to the factory default settings. Refer to the LAN Modem User Guide. User guides are available in the Manuals section of OfficeConnect Network Assistant.

  3. If this does not solve the problem, contact Technical Support. Refer to your user guide for technical support information.
