Setting Up a PC

This is the first of four installation sections, taking you step by step through setting up a network.

This section will explain how to:

  • Prepare your PC or laptop for connection to a network
  • Upgrade drivers should you need to

If you are new to networking, please read Networks Explained before tackling the installation sections.

Note: When setting up your PC, at various points you may need to restart the PC. If possible, run this OfficeConnect Network Assistant CD from a different computer to the one you are setting up.

Click on one of the left hand icons to begin.
