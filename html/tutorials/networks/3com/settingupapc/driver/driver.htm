1 Overview

1.1 This subsection describes how to change the driver for a PC's NIC or a laptop's PCMCIA card.

You may want to do this if you have a new driver with improved features. The new driver may be supplied with an upgrade (such as a Windows 95 or Windows 98 upgrade) or can be downloaded from the Internet if you have an Internet connection.

To change the driver you require the new driver software and any accompanying documentation.

2 Changing the driver

2.1 To change the driver:

Power on the PC or laptop and start Windows 95 or Windows 98.

2.2 From the Start menu choose Settings and then Control Panel.

2.3 In the Control Panel window, double-click System.

2.4 In the System Properties dialog box, select the Device Manager tab.

2.5 In the listbox, double-click Network adapters. The Network adapters item should expand to show the name of the NIC or PCMCIA card that is installed. Select the NIC or PCMCIA card and click Properties.

2.6 Select the Driver tab and then click Update Driver.

2.7 In the Update Device Driver Wizard dialog box, with Windows 95 select No, select driver from list, or with Windows 98 select Display a list of all the drivers in a specific location…. Click Next>.

2.8 Locate the driver (if copied onto the PC or laptop's hard drive) or click Have Disk and provide the new driver disks (if necessary refer to the documentation that accompanies the driver disk and the NIC and PCMCIA card). You may be asked to select the appropriate driver from a list.

2.9 Windows asks you to insert your Windows 95 or Windows 98 CD or various Windows disks. Provide them as required and remove after use.

2.10 When finished, click Finish and restart the PC or laptop.

You may need to re-enter some of the PC or laptop's network information to make the new driver ready for use with the network.

To do this:

  • For a PC, go to Making a PC ready for network use.
  • For a laptop, go to Making a laptop ready for network use.
