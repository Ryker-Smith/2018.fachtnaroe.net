1 Overview

1.1 To set up a network you must have a network interface card (NIC) in each PC. A NIC (sometimes referred to as an adapter card) is a circuit board that you install into the PC so it can connect to the network.

Alternatively, you may wish to use a USB Network Interface to connect a Windows 98 PC to your network.

This subsection describes how to install a NIC in the PC and how to configure the PC for use with the network.

If you are using the USB Network Interface you must first install it following the instructions in your installation guide, and then go to Making a PC ready for network use.

We assume that Windows 95 or Windows 98 is already installed on the PC.

1.2 Now read the following:

  • If you need to set-up a Laptop as opposed to a PC, go to Adding a Laptop.
  • If the PC does not have a NIC in it you need to install a NIC. Go to Installing the NIC.
  • If the PC has a NIC in it go to Making a PC ready for network use.
  • If the PC is a 'network ready' PC (a PC with a built-in network card) go to Making a PC ready for network use.
  • If you are not sure whether there is a NIC already in the PC look on the back to see if there is a suitable RJ-45 connector. If there is you may well have a NIC, so go to Making a PC ready for network use.
  • If you want to upgrade the NIC's driver go to the subsection Changing the Driver.

2 Installing the NIC

2.1 To install your network adapter card (NIC) you should have:

  • The manual that accompanies the PC.
  • The NIC itself and accompanying CDs, disks and manuals.
  • A suitable screwdriver for removing the PC's cover and securing the NIC in place.
  • Windows 95 or Windows 98 CD-ROM or disks.

Carefully read the instructions that accompany your card because the slots and connections vary.

2.2 To install the NIC into the PC:

1. Ensure that you know the exact name for the NIC. When configuring the NIC there may be several similar driver names and selecting the correct one is important.

2. Power off the PC and disconnect its power cable.

3. Refer to the PC's manual for safety information and instructions on removing the PC's cover.

2.3 Remove the cover of the PC and insert the NIC into the PC.

Make any necessary connections, referring to the instructions in the NIC's manual. Avoid touching the components on the PC and NIC as they can be damaged easily. If there is more than one slot of the required type, insert the NIC into any of these.

It is very important that the NIC is fully inserted into the appropriate slot. This may mean using a little force when pushing the NIC in place.

2.4 Replace the top cover of the PC and reconnect its power cable and any other cables that you may have disconnected when removing the cover.

Power on the PC and restart Windows 95 or Windows 98.

2.5 When the PC and Windows 95 or Windows 98 are starting up, the NIC may be detected automatically.

If so, configuration screens should appear to allow you to configure it.

  • If the NIC is not detected, go to Configuring the NIC.
  • If the NIC is detected automatically and configuration screens appear for it, follow the configuration instructions on the PC screen and those that are in the NIC's manual.

Other new hardware may be detected at the same time as the NIC. Refer to the documentation accompanying that hardware for information on installing it.

2.6 With Windows 95, when configuring the NIC you may be asked for:

The Computer name
This is the name that will identify the PC on the network. It can be any name (such as 'Fred's PC') but it must be unique from other PCs and laptops on the network. We recommend that you keep a record of all the names on the network.

The Workgroup name
This is the name for the network. It can be any name but it must be the same as the workgroup name entered for other PCs and laptops on the network. We recommend that you keep a record of the workgroup name.

Windows 98 will assume the Computer and Workgroup names entered when Windows 98 was installed. To check the names, continue to section 2.7. If you are not running Windows 98, or do not want to check the names, go straight to section 2.8 of this topic.

2.7 To check the Computer and Workgroup names in Windows 98:

1. From the Start menu, choose Settings and then Control Panel.

2. In the Control Panel window, double-click Network.

3. In the Network dialog box, select the Identification tab.

4. Check the names are suitable:

Computer name
This is the name that will identify the PC on the network. It can be any name (such as 'Fred's PC') but it must be unique from other PCs and laptops on the network. We recommend that you keep a record of all the names on the network.

Workgroup name
This is the name for the network. It can be any name but it must be the same as the workgroup name entered for other PCs and laptops on the network. We recommend that you keep a record of the workgroup name.

2.8 When the configuration process is completed, restart the PC and Windows as prompted and go to Making a PC ready for network use.

If a DHCP error message appears when the PC and Windows 95 have been restarted go to the Troubleshooting section.

2.9 To make the PC ready for network use:

Start Windows 95 or Windows 98.

If the NIC was automatically detected and the Computer name and Workgroup have already been specified, Windows may ask you to log in. If you have logged on into this PC before, enter your User name and Password as normal. If this is the first time you have logged on to this PC, enter a new User name in the User name field (for example, your name). This name should be unique to the network. To set a password, simply type the desired password into the Password field. Click OK. Windows asks you to confirm the password. Retype it and click OK.

3 Configuring the NIC

3.1 You need to complete this section only if the NIC was not automatically detected as described in the previous section Installing the NIC.

To configure the network adapter card (NIC) you will need:

  • The CDs, disks and manuals that came with the NIC
  • Windows 95 or Windows 98 CD-ROM or disks

The process uses a 'Wizard' that is part of your Windows 95 or Windows 98 software.

A Wizard is a series of screens in which you are asked to make choices or indicate where software can be found. If you follow each step carefully the right software (or 'driver') for your NIC will be installed.

Note: The detailed appearance of the Wizard screens may differ depending on the version of Windows you are running.

3.2 To configure the NIC:

1. Power on the PC and start Windows 95 or Windows 98, if necessary.
2. From the Start menu, choose Settings and then Control Panel.

3.3 In the Control Panel window, double-click Add New Hardware.

3.4 In the Add New Hardware Wizard dialog box, click Next>.

If you are using Windows 98, click Next> again.

3.5 Select Yes for the Windows new hardware search option, and click Next>.

3.6 A message appears warning that the detection process may take some time. Click Next>.

The Add New Hardware Wizard tries to detect the NIC. This process may take a few minutes depending on the speed of the PC.

3.7 If the NIC is detected:

Click Finish.

The Add New Hardware Wizard asks you to insert the Windows 95 or Windows 98 CD or various Windows 95 disks. Provide them as required and remove after use.

When finished, restart the PC as prompted.

  • If the NIC is not detected, you must add the NIC manually.
    Click Next.

3.8 You only need to continue with the following manual process if the NIC was not detected. To add the NIC manually:
In the list box, select Network Adapter and click Next.

3.9 In the list boxes, select the manufacturer and model of the NIC you have installed. If the manufacturer or model is not listed, click Have Disk and provide the disks that are supplied with the NIC (refer to the NIC's manual if necessary).

3.10 Windows asks you to insert your Windows 95 or Windows 98 CD or various Windows disks. Provide them as required.
When finished, click Finish and restart the PC.

If, in the future, you need to upgrade the NIC’s driver (for example, to a new driver with improved features), go to the subsection Changing the Driver.

4 Making a PC ready for network use

4.1 To make the PC ready for network use:

Start Windows 95 or Windows 98.

If the NIC was automatically detected and the Computer name and Workgroup have already been specified, Windows may ask you to log in. If you have logged on into this PC before, enter your User name and Password as normal. If this is the first time you have logged on to this PC, enter a new User name in the User name field (for example, your name). This name should be unique to the network. To set a password, simply type the desired password into the Password field. Click OK. Windows asks you to confirm the password. Retype it and click OK.

4.2 Look to see that the desktop contains an icon called Network Neighborhood:

  • If the icon is not present the NIC is not installed correctly. Go to the Network Neighborhood icon is absent from the desktop screen section.
  • If the icon is there, then the PC has detected the physical installation of your NIC. You may need to check the software configuration of you NIC if you experience problems later on.

4.3 You now need to configure the PC so that it can be recognized on the network.

From the Start menu, choose Settings and then Control Panel.

4.4 In the Control Panel window, double-click System.


4.5 In the Systems Properties dialog box select the Device Manager tab.

4.6 In the listbox double-click Network adapters.
The Network adapters item should expand to show the name of the NIC that you have installed.

4.7

  • If the NIC is detected and the icon for it does not have a yellow circle and exclamation mark (!) on it, the NIC is configured correctly.
  • If the NIC is not detected (or the icon for it has a yellow circle and an exclamation mark (!) on it), the NIC is not configured correctly.

If the NIC was automatically detected and the Computer name and Workgroup have already been specified, go straight to section 4.11 of this topic.

4.8 In the Control Panel window double-click Network.

Note: the following section only applies if you have a network with some PCs running Windows 95, and some PCs running Windows 98. If all your PCs are running Windows 95, or if all your PCs are running Windows 98, go straight to section 4.9 of this topic.

By default, Windows 95 and Windows 98 install different network protocols. Windows 95 installs NetBEUI by default, and Windows 98 installs TCP/IP by default.

If different network protocols are used, Windows 95 PCs and Windows 98 PCs will not be able to communicate. We suggest that you configure your Windows 95 PCs to use TCP/IP. To do this, refer to the section Setting up IP, then return here and continue.

In the Network dialog box, select the Configuration tab. The network protocols in use are shown in the listbox, identified by the symbol. You will need to check the correct protocol is used for each of the PCs in your network.

4.9 Windows 98 will assume the Computer and Workgroup names entered when Windows 98 was installed. If you do not wish to change the names, click Cancel and go straight to the next page.

In the Network dialog box, select the Identification tab.

4.10 In the Computer name field, type a descriptive name for the PC.

This is the name that will identify the PC on the network. It can be any name (such as 'Fred's PC') but it must be unique from the other PCs and laptops on the network. We recommend that you keep a record of the names given to all the PCs and laptops on the network.

In the Workgroup field, type a name for your network.

This is the name for the network. It can be any name but it must be the same as the workgroup name entered for the other PCs and laptops on your network. We recommend that you keep a record of the workgroup name.

Click OK.

4.11 In the Control Panel window, double-click Network.

4.12 In the Configuration tab (which displays by default), click File and Print Sharing (this needs to be configured to utilize your network).

4.13 In the File and Print Sharing dialog box, select both of the checkboxes, and click OK.

4.14 In the Network dialog box, ensure the Primary Network Logon is set to Client for Microsoft Networks. Click OK. Restart the PC.

4.15 In the future when you power on the PC and start Windows 95 or Windows 98, you will be asked to log in. You must enter your User name in the User name field (and specify a password if one has been used previously), then click OK to log in to the network.

To set a password (if one is not already set up) simply type in the desired password into the Password field. When OK is clicked, Windows 95 or Windows 98 asks you to confirm the password. Retype it and click OK.

This PC is now ready to be installed on your network.
