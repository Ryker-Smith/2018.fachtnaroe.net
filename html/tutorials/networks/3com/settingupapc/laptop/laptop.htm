1 Overview

1.1 Every laptop you want to use on the network has to have a network card installed. This subsection describes how to install a PCMCIA card (sometimes known as a PC card) in the laptop and how to configure the laptop for use with the network.

We assume that Windows 95 or Windows 98 is already installed on the PC.

  • If the laptop does not have a PCMCIA card, you need to insert one. Go to Installing the PCMCIA card.
  • If the laptop does have a PCMCIA card and suitable connector go to Making a laptop ready for network use.
  • If you are not sure whether the laptop already has a PCMCIA card in it, look in the card slots. If there is one and you have an adapter cable for it, go to Making a laptop ready for network use.
  • If you want to upgrade the PCMCIA card's driver go to the subsection Changing the Driver.

2 Installing the PCMCIA card

2.1 To install your PCMCIA card you should have:

  • The manual that accompanies your laptop.
  • The PCMCIA card and accompanying disks, CDs and manuals.
  • An appropriate adapter cable for the PCMCIA card to connect it to the network.
  • Your Windows 95 or 98 CD-ROM or disks.

Some laptops do not have built-in disk drives and instead use their PCMCIA card slot to attach external disk drives to the laptop. If the laptop into which you are inserting the PCMCIA card is one of these, you may need to copy various configuration files (from the disk supplied with the PCMCIA card) onto the laptop before inserting the PCMCIA card. Refer to the PCMCIA card's manual.

2.2 For a successful installation Card Services must be enabled on the laptop. To check this:
1. Power on the laptop and start Windows 95 or Windows 98.
2. From the Start menu choose Settings and then Control Panel.

2.3 In the Control Panel window double-click PC Card (PCMCIA).

  • If the PC Card (PCMCIA) Properties dialog box appears, Card Services is enabled. (Card Services is required to detect insertion of a PCMCIA card into a laptop.)
  • If the PC Card (PCMCIA) Wizard appears, Card Services is not enabled. In the Wizard select No and click Next>. Click Finish. Restart the laptop and Windows as prompted.

2.4 To insert the PCMCIA card into the laptop:

1. Power off the laptop and disconnect its power cable.

2. Refer to the PCMCIA card's manual to check if the adapter cable has to be connected before the card is inserted into the laptop. If so, connect the cable.

3. Insert the PCMCIA card into the laptop (ensuring it is the right way round).

4. Power on the laptop and start Windows.

 

2.5 When the laptop and Windows 95 or Windows 98 are starting up, the PCMCIA card may be detected automatically. If this is the case, configuration screens will appear to allow you to configure it; go on to section 2.7.

If the PCMCIA card is not detected:

1. Ensure that the card is inserted correctly.
2. Ensure that Card Services is enabled on the laptop. To check Card Services follow the instructions on section 2.2 and section 2.3 of this topic.

If the PCMCIA card is still not detected, go to the Troubleshooting section of this CD.


2.6 If the PCMCIA card is detected after restarting the laptop go to section 2.7.

If the PCMCIA card is detected automatically and configuration screens appear for it, follow the configuration instructions on the PC screen and those that are in the PCMCIA card's manual.

Other hardware may be detected at the same time as the PCMCIA card (if that hardware had not been previously installed). Refer to the documentation accompanying that hardware for information on installing it.

2.7 When configuring the PCMCIA card you may be asked for:

The Computer name
This is the name that will identify the laptop on the network. It can be any name (such as 'Jane's laptop') but it must be unique from other PCs and laptops on the network. We recommend that you keep a record of all the names on the network.

The Workgroup name
This is the name for the network. It can be any name but it must be the same as the workgroup name entered for other PCs and laptops on the network. We recommend that you keep a record of the workgroup name.

2.8 When the configuration process is completed, restart the laptop and Windows as prompted and go to Making a laptop ready for network use.

If a DHCP error message appears when the PC and Windows has been restarted go the Troubleshooting section.

3 Making a laptop ready for network use

3.1 To make a laptop ready for network use:

Start Windows 95 or 98.

If the NIC was automatically detected and the Computer name and Workgroup have already been specified, Windows may ask you to log on. If you have logged on to this laptop before, enter your User name and Password as normal. If this is the first time you have logged on to this laptop, enter a new User name in the User name field (for example, your name). This name should be unique to the network. To set a password, simply type the desired password into the Password field. Click OK. Windows asks you to confirm the password. Retype it and click OK.

3.2 Look to see that the desktop contains an icon called Network Neighborhood:


  • If the icon is present the PCMCIA card is installed correctly. Go to the section 3.3.
  • If the icon is not present the PCMCIA card is not installed correctly. Go to the Network Neighborhood icon is absent from the desktop screen section.

3.3 You now need to configure the laptop so that it can be recognized on the network.

From the Start menu, choose Settings and then Control Panel.

3.4 In the Control Panel window, double-click System.

3.5 In the Systems Properties dialog box select the Device Manager tab.

3.6 In the listbox double-click Network Adapters.
The Network adapters item should expand to show the name of the PCMCIA card that you have inserted.


3.7 If the PCMCIA card is detected (and the icon for it does not have a yellow circle and exclamation mark (!) on it), the PCMCIA card is configured correctly. In the System Properties dialog box click Cancel and continue with this topic.

If the PCMCIA card is not detected (or the icon for it has a yellow circle and exclamation mark (!) on it), the PCMCIA card is configured incorrectly. Go to the Troubleshooting section.

3.8 In the future when you power on the laptop and start Windows, you will be asked to log in.

You must specify the user name in the User name field (and specify a password if one has been used previously) and click OK to log in to the network.

To set a password (if one is not already set up) simply type in the desired password into the Password field on the Enter Network Password dialogue box. Click OK, Windows asks you to confirm the password. Retype it and click OK.

3.9 This laptop is now ready to be installed on your network.
