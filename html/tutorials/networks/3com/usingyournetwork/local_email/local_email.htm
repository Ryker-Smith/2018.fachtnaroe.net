1 Overview

1.1 This subsection describes how to set up and use the Windows Messaging system provided with Windows 95. This allows users of the network to send e-mails to each other.

Note that a mail server is not supplied with Windows 98. If you want to set up e-mail under Windows 98, you will need to purchase suitable software from your supplier, or use the e-mail software supplied with Internet Explorer or Netscape Navigator.

One of the PCs on the network needs to be configured as a 'post office' that stores all the e-mails.

To use e-mail, users must have an e-mail account set up at the post office. The user's PC or laptop periodically checks the post office for new e-mail. Any user can have an e-mail account, including the user of the PC that is set up as the post office.

The installation steps only describe how to set up e-mail for the peer-to-peer network that you have created. For Internet e-mail, you will need an Internet connection and appropriate Internet software with e-mail capabilities. Contact your supplier for details.

2 Setting up the post office

2.1 It is recommended that you use a PC for the post office instead of a laptop, because the machine must be present and powered on at all times for the e-mail system to work.

To set up a post office on one of the PCs, you need to:

  • Create a shared post office folder.
  • Install Windows Messaging.
  • Set up Postoffice Administrator.

2.2 To create a shared post office folder, carry out the following steps:

Start Windows 95.

Create a new post office folder, for example 'C:\postoffice', and keep a record of this. (This is called a pathname.)

To do this, either:

  • In the Windows 95 desktop, double-click the My Computer icon and create the folder.
  • Or use Windows Explorer to create the folder.

To start Windows Explorer, from the Start menu choose Programs and then Windows Explorer. For information on how to use Windows Explorer, refer to the Windows 95 manual and help system.

2.3 Right-click on the post office folder you have created (using the right mouse button) and select Sharing. The Properties dialog box appears with the Sharing tab displayed.

2.4 In the Sharing tab, select Shared As and specify access type as Full.

2.5 Now click OK to create your shared post office folder.

2.6 To install Windows Messaging carry out the following steps:

From the Start menu, choose Settings and then Control Panel.

2.7 In the Control Panel window, double-click on Add/Remove Programs.

2.8 In the Add/Remove Programs Properties dialog box, select the Windows Setup tab.

2.9 In the listbox, select Windows Messaging (do not click the checkbox) and then click Details.

2.10 In the Messaging dialog box, select Windows Messaging and then Microsoft Mail Services, and click OK.

2.11 In the Windows Setup tab, click OK to get Windows 95 to install Windows Messaging software.

2.12 The Inbox Setup Wizard is launched automatically. This is not required just yet, so click Cancel and then Yes to confirm the cancel.

2.13 To set up Postoffice Administrator carry out the following steps:

From the Start menu, choose Settings and then Control Panel.

2.14 In the Control Panel window, double-click on Microsoft Mail Postoffice.

A window appears welcoming you to the Postoffice Admin utility.

2.15 Select Create a new Workgroup Postoffice and click Next>.

2.16 Select Browse and locate the Postoffice folder that you created earlier. Click OK and then Next>.

A window appears showing the pathname for the new workgroup postoffice.

Keep a record of the pathname and click Next>.

2.17 Specify the mailbox (mailbox name), username and password for the post office administrator (the user that is responsible for using the post office to set up new e-mail accounts).

Keep a record of these details and click OK.

2.18 A window appears, warning that the folder must be shared. This has already been done. Click OK.

2.19 In the Control Panel window, double-click on Microsoft Mail Postoffice.

2.20 Select Administer an existing Workgroup Postoffice and click Next>.

2.21 Specify the pathname for the postoffice folder that was created on page 16. The pathname may already be in the field. Then click Next>.

2.22 Enter the administrator's mailbox name and password and click Next> to start Postoffice Manager.

The administrator is now ready to create e-mail accounts for the users of the network.

3 Creating e-mail user accounts

3.1 To create new e-mail accounts at the post office PC, do the following:

  • If you already have Postoffice Manager open, go straight to page 5 of this topic.
  • If you do not have Postoffice Manager open, in the Control Panel window, double-click on Microsoft Mail Postoffice.

3.2 Select Administer an existing Workgroup Postoffice and click Next>.

3.3 The pathname for the postoffice folder should already be entered. Click Next>.

3.4 The administrator's mailbox name should already be entered.

Enter the password and click Next> to start Postoffice Manager.

3.5 Click Add User.

3.6 Enter the new user's name (user name), mailbox (mailbox name) and password. Make a note of these details to give to the new user, and suggest that they change their password when e-mail has been set up on their PC or laptop. Click OK.

All mailbox names must be different. Passwords in the e-mail system are not case-sensitive.

3.7 Create new e-mail accounts for other new users by performing these steps for each user, and then click Close.

4 Setting up e-mail on a PC or laptop

4.1 To set up e-mail on the user's PC or laptop carry out the following steps:

From the Start menu, choose Settings and then Control Panel.

4.2 In the Control Panel window, double-click on Add/Remove Programs.

4.3 In the Add/Remove Programs dialog box, select the Windows Setup tab.

4.4 In the listbox, select Windows Messaging (do not click the checkbox) and then click Details.

4.5 In the Messaging dialog box, select Windows Messaging and then Microsoft Mail Services, and click OK.

4.6 In the Windows Setup tab, click OK to get Windows 95 to install the Windows Messaging software.

4.7 The Inbox Setup Wizard is launched automatically, asking you if you have used Windows Messaging before.

Select No and then click Next> twice.

4.8 If this is the PC that has the post office set up on it, select Browse, locate the post office folder, and then click OK and Next>.

4.9 If this is not the PC or laptop that has the post office set up on it, locate the folder.

To do this, double-click the icon for the PC or laptop that has the post office folder on it and select the post office folder. Click OK and then Next>.

4.10 In the listbox, select the user name of the e-mail account for this user (this is the e-mail account that was set up on the post office). Click Next>.

4.11 Type the password for the e-mail account and click Next>.

4.12 Click Finish. The user is now ready to use e-mail.

5 Using e-mail

5.1 This topic describes how to access and organize your e-mail and send, receive and store messages. For more advanced use of e-mail, refer to its Microsoft Mail Help.

A user can access and use e-mail when:

  • An account has been created for them on the post office.
  • E-mail has been set up on their PC or laptop.

If this has not been done, refer to Creating e-mail user accounts and Setting up e-mail on a PC or laptop.

5.2 To start e-mail, in the Windows 95 desktop, double-click the Inbox icon.

5.3 To change the password, carry out the following steps:

From the Tools menu, choose Microsoft Mail Tools then Change Mailbox Password.

5.4 Specify the current password and the new password twice in the appropriate fields.

Click OK.

5.5 To use the mail folders:

To view the mail folder names (by the side of the currently viewed mail folder), from the View menu choose Folders.

5.6 The Inbox folder contains e-mail that has been sent to you, with new e-mail in bold lettering.

5.7 Windows Messaging periodically checks the post office for new e-mail. However, to manually check for new e-mail, from the Tools menu choose Deliver Now.

5.8 To open an e-mail:

Double-click the e-mail in the mail folder.

To open a file that has been sent with the e-mail, known as an attached file, double-click the file icon in the message. The file opens in the application that it is associated with.

5.9 To send an e-mail, carry out the following steps:

Click the New Message icon.

5.10 Specify the users that you are sending the e-mail to. To do this:

Click To.

The Address Book dialog box appears. In the left listbox, the e-mail users are shown. Select the users that you want to send the message to, and click To or Cc as required.

When finished, click OK.

5.11 In the Subject field, specify a title for the e-mail.

5.12 In the Message area (below the Subject field), type your message.

5.13 To attach a file to the message (a copy of this file will be sent with the e-mail):

Click the Paper clip icon.

Select Browse, locate the file and click OK.

Click OK.

5.14 Click the Send icon.

5.15 To store e-mail in mail folders carry out the following steps:

Select the messages to be moved to the mail folder.

Click the Move item icon.

5.16 Select the mail folder that the e-mail is to be moved to, and click OK.

5.17 If you need to create a new mail folder, click New folder, specify the name of the new mail folder, and click OK.
