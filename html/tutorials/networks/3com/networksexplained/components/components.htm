1 Overview

1.1 A small network generally consists of:

  • PCs, cabling and peripherals such as printers and print servers
  • Equipment that allows your PCs and peripherals to communicate such as an OfficeConnect hub or switch
  • Modems, LAN modems and routers
  • A network operating system (NOS) such as Windows 95 or 98

2 NICs

2.1 All PCs require network interface cards (NICs) to make them ready for network use. Some are supplied with NICs already in them. When choosing a NIC (also known as an adapter card) for a PC you should consider:

  • The speed of your hub, switch, or print server— Ethernet (10Mbps) or Fast Ethernet (100Mbps).
  • The type of connection you need—RJ-45 for twisted pair or BNC for coaxial cable.
  • The type of NIC connector available inside your PC—ISA or PCI.

2.2 Connection Speed

An Ethernet NIC must be used with an Ethernet hub or switch, and a Fast Ethernet NIC must be used with a Fast Ethernet hub or switch.

If you are connecting your PC to a dual speed device that supports both 10 and 100Mbps, you can use either a 10Mbps or a 100Mbps NIC. A port on a dual speed device automatically adjusts its speed to match the highest speed supported by both ends of the connection. For example if the NIC only supports 10Mbps the port on a dual speed hub that is connected to that NIC becomes a 10Mbps port. If the NIC supports 100Mbps the port speed on the hub will be 100Mbps.

In a similar way, if you have a dual speed NIC, you can connect it to either a 10Mbps Ethernet hub or a 100Mbps Fast Ethernet hub. The dual speed NIC will adjust its speed to match the highest speed supported by both ends of the connection.

Note: Dual speed devices are also known as auto-negotiating, auto-sensing, or 10/100 device.

2.3 Connection Type

If you are setting up a network that uses twisted pair cabling you will need a NIC with an RJ-45 connector.

2.4 ISA and PCI Connectors

There are two common types of NIC connectors for PCs:

  • ISA (Industry Standard Architecture) sockets are about 14cm long.
  • PCI (Peripheral Component Interconnect) sockets are used in all Pentium desktop PCs. PCI sockets provide a much higher performance than ISA. PCI sockets are about 9cm long.

Refer to your PC's user guide to find out which connector type is available on your PC.

2.5 Specialized NICs

In certain cases you may need specialized NICs. For example, if you have a laptop you will require a PCMCIA card.

When choosing a PCMCIA card you should consider:

  • The speed of your hub, switch, or print server—Ethernet (10Mbps) or Fast Ethernet (100Mbps).
  • The type of connection you need—RJ-45 for twisted pair or BNC for coaxial cable.

If you have a USB port, you may consider using a USB Network Interface.

3 Hubs and switches

3.1 Hubs and switches are used to connect your PCs, printers and other devices. Hubs differ from switches in the way that they pass on the network traffic.

3.2 The term “hub” is sometimes used to refer to any piece of network equipment that connects PCs together, but actually serves as a repeater. This is because it passes on or repeats all the information it receives, to all its ports.

Hubs can be used to extend a network. However, this can result in a lot of unnecessary traffic because the same information is sent to all devices on a network.

Hubs are fine for small networks, but heavily used networks may require additional networking equipment such as a switch which will reduce unnecessary traffic.

3.3 Switches use the address information in each packet to control the flow of network traffic. By monitoring the packets it receives, a switch learns which devices are connected to its ports and then sends on the packets to the appropriate ports only.

A switch reduces the amount of unnecessary traffic because information received on a port is only forwarded to the device with the appropriate destination address, unlike a hub, which forwards to all ports.

3.4 Switches and hubs are often used in the same network. The hubs extend the network by providing more ports, and the switches divide the network into smaller, less congested sections.

3.5 In a small network, hubs can easily cope with the network traffic generated. When the network reaches about 25 users, you may need to cut down unnecessary traffic. A suitable switch, such as one from the OfficeConnect range, will divide the network to achieve this.

Some hubs have Network Utilization LEDs, which you can use to view the amount of traffic on the network. If the traffic is constantly high, you may need to divide up the network using a switch.

3.6 When adding hubs to the network there are rules you need to know about the number of hubs you can connect together. Switches can be used to extend the number of hubs in your network. Refer to the Network Rules section of this CD for more information.

4 Modems, Routers and Firewalls

4.1 A modem is a device that connects directly to your computer and uses the telephone line to dial remote sites, such as an online service or an ISP. A modem's essential task is to convert the digital data required by your computer into analog signals for transmission over the telephone line and vice versa.

The speed at which a modem connects is measured in Kilobits per second (Kbps). Most modems used today connect at speeds ranging from 28.8Kbps to 56Kbps. Modems are also defined according to their ITU standard. For example, a modem that is capable of download speeds of up to 56Kbps is referred to by the designation V.90.

4.2 A LAN Modem is similar to a stand-alone modem in that it is a device that uses the telephone line to dial remote sites. 3Com's OfficeConnect LAN Modem, for example, is essentially an ISDN or analog router with a built-in Ethernet hub, allowing users to share phone lines and modem connections. A LAN Modem is connected directly to each attached computer's Ethernet network port, allowing for potentially faster data transfers than stand-alone modems.

4.4 In a network a firewall is a node set up as a barrier to prevent unwanted traffic crossing from one segment to another. Firewalls also improve network security, and may serve as a barrier between connected public and private networks. A firewall may be implemented in a router or it may be a network device specialised for this purpose.

By using a firewall you can prevent access to inappropriate material, keep a log of which sites are being accessed most frequently and analyse how much bandwidth your Internet connection is using. 5 Network Operating Systems

5.1 Your computer has a Network Operating System that allows it to offer services to other users over the network.

There are different types of Network Operating Systems. For example, Microsoft has produced a number of operating systems including Windows 95, Windows NT, and more recently the Windows 98 operating system. These operating systems communicate with other devices on your network using a set of rules. These rules are known as Protocols.

An operating system can support several protocols, but only devices using the same protocol can communicate with each other.

When you connect your computer to the network (using a NIC, PCMCIA card or modem) the computer will automatically associate a protocol with that device. The protocol associated with the device by default will depend on the operating system installed on your computer.

For example, Windows 95 installs the NetBEUI protocol by default and Windows 98 installs the TCP/IP protocol by default.

If you have some computers using NetBEUI while other use TCP/IP, you will have two different networks. The computers using NetBEUI (typically with Windows 95 operating systems) will only be able to recognise and communicate with other computers using NetBEUI. Computers using TCP/IP (typically with Windows 98 operating systems) will only be able to communicate with other computers using TCP/IP.

To solve this problem, you must make sure that all computers on the network use the same protocol.

We recommend that you configure your computers to use TCP/IP if:

  • You require Internet access either now or in the future.
  • You will be using software that requires TCP/IP. For example, many computer games now require TCP/IP.
  • The majority of your computers already have Windows 98 installed.

Setting up TCP/IP is described in Setting up IP within the Using Your Network section of this tutorial.
