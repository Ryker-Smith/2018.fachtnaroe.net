1 Overview

1.1 A print server is a good way to share a printer on a network by connecting to virtually any printer with a parallel port. Print servers permit a printer to be located anywhere on a network. They act as the server receiving information and passing it on to the printer so that jobs can be printed. Using a print server eliminates the need to attach a printer via the PC or file server and results in faster, more flexible printing.

Hewlett-Packard (HP) has joined 3Com's OfficeConnect family of networking products by providing HP JetDirect external print servers which clip and integrate with the rest of the OfficeConnect products.

Before: No Print Server; printer attached to one PC; PCs attached to hub.

After: Printer attached to Print Server; PCs attached to hub.
