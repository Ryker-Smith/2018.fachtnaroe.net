1 Overview

1.1 This subsection describes how to add and access a local printer. A local printer is a printer that is connected directly to one of the PCs on the network, as opposed to a network printer which is connected through a print server.

These installation steps describe how to set up a local printer but not how to set up a network printer. There are many different ways to connect and configure a network printer and you should refer to the documentation that accompanies the printer and/or print server (see Installing a Printer Using a Print Server).

1.2 To set up a local printer you need:

  • The printer, power cable, printer cable (to connect the printer to the PC) and any accompanying CDs, disks and manuals.
  • Some paper for the printer and any other consumables such as a toner cartridge.
  • Windows 95 or Windows 98 CD or disks.

If the printer is not connected to a PC, go to Setting up a local printer.

If the printer is already connected and usable by the PC to which it is connected go to Sharing a local printer.

2 Setting up a local printer

2.1 To set up a local printer:

  • Follow the instructions in the printer's manual to physically connect the printer to one of the PCs.
  • Power on the printer, put paper in it and ensure that it has everything required to print (such as a toner cartridge).
  • Ensure that the printer is online (ready to print).
  • Power on the PC and start Windows 95 or Windows 98.

2.2 The printer may be detected automatically, in which case configuration screens should appear to allow you to configure it.

If the printer is detected, configure it following the instructions in the printer's manual. Print a Windows test page to test the configuration.

If a test page is printed correctly the printer is ready for use. Go to Sharing a local printer.

If the test page does not print or prints incorrectly, use the Windows Printer Troubleshooter and the printer's manual to get the printer working.

If you have any problems with the quality or alignment of the printed page, refer to the printer's manual for information on correcting the problem.

If the printer is not detected, go through the following steps. If the printer is detected, go straight to Sharing a local printer.

2.3 From the Start menu, choose Settings and then Printers.

2.4 In the Printers window, double-click Add Printer.

2.5 In the Add Printer Wizard dialog box, click Next>.

2.6 Select Local printer for the printer type and click Next>.

2.7 In the list boxes, select the manufacturer and model of the printer you have connected. If the manufacturer or model is not listed, click Have Disk and provide the disks that are supplied with the printer (refer to the printer's manual if necessary).

Note: The following screens may differ slightly according to your Windows and printer setup.

2.8 In the list box, select the port to which the printer is connected and click Next>. Most printers connect to the PC's parallel port which is listed as 'LPT1'.

2.9 Specify a name for the printer (if a different name is desired). This is the name that is used to identify the printer in Windows. It can be any name you like.

Click Next>.

2.10 To print a test page from the Printer Wizard dialog box, click Finish.

Windows asks you to insert your Windows 95 or Windows 98 CD or various Windows disks. Provide them as required. Windows copies the appropriate printer drivers from the CD or disks. Remove the CD or disks after use.

2.11 When finished, the printer should print a test page (unless you selected No for the print test page option):

  • If a test page is printed correctly, the printer is ready for use. Click Yes in the Add Printer Wizard dialog box. Go to Sharing a local printer.
  • If the test page does not print or it prints incorrectly, click No in the Add Printer Wizard dialog box. Windows starts the Windows Printer Troubleshooter. Use the Printer Troubleshooter and the printer's manual to get the printer working.

If you have any problems with the quality or alignment of the printed page, refer to the printer's manual for information on correcting the problem.

3 Sharing a local printer

3.1 To make a local printer shared:

Ensure the printer and PC are powered on. Start Windows 95 or Windows 98 (if necessary).

3.2 At the PC to which the printer is connected, from the Start menu, choose Settings and then Printers.

3.3 In the Printers window, there should be an icon for the printer:

  • If there is an icon for the printer, go to the next page.
  • If there is no icon for the printer, the printer is not set up correctly. Go to Setting up a local printer.

3.4 Right-click on the printer icon and select Sharing. The Printer Properties dialog box appears with the Sharing tab displayed.

3.5 In the Sharing tab, select Shared As.

If required, specify:

  • Shared name— the name that is used to identify the printer on the other PCs and laptops that access it.
  • Password—if you want to restrict access to the printer, you can give it a password that must be entered on the other PCs and laptops that are to access it. Ensure you do not forget the password.

Click OK.

3.6 In the Printers window, the icon for the printer should show a hand underneath the printer, to identify that it has been made shared. The printer can now be accessed by other PCs and laptops on the network.

4 Printing over the network

4.1 To access a local printer on the network from another PC or laptop:

  • Ensure that the printer and the PC to which it is connected are powered on.
  • Ensure that the PC or laptop that is to share the printer is powered on and running Windows 95 or Windows 98.

4.2 From the Start menu, choose Settings and then Printers.

4.3 In the Printers window, double-click Add printer.

4.4 In the Add Printer Wizard dialog box, click Next>.

4.5 Select Network printer for the printer type and click Next>.

4.6 Browse for the printer:

Click Browse.

In the Browse For Printer dialog box, locate the printer. To do this, double-click the icon for the PC to which the printer is connected, highlight the printer and click OK.

Note: If you wish to print from MS-DOS-based programs make sure that you have checked the Yes option.

In the Add Printer Wizard dialog box, click Next>.

4.7 Change the name of the printer (if required) and click Next>. This is the name that is used to identify the printer on this PC or laptop.

4.8 To print a test page from the Printer Wizard dialog box, click Finish.

Windows copies the appropriate printer drivers (over the network) from the PC to which the printer is connected.

4.9 When finished, the printer should print a Windows test page (unless you selected No for the print test page option):

  • If a test page is printed correctly, the printer is ready for use by this PC or laptop. Click Yes in the Add Printer Wizard dialog box. Go to the next page.
  • If the test page does not print or it prints incorrectly, click No in the Add Printer Wizard dialog box. Windows starts the Windows Printer Troubleshooter. Use the Printer Troubleshooter and the printer's manual to get the printer connection working.

4.10 If you have more than one printer and you want to make this printer the default printer for this PC or laptop:

From the Start menu, choose Settings and then Printers.

4.11 Right-click on the printer icon and select Set As Default.

The printer is now set as the default printer, and is used when printing from this PC or laptop.

The printer can now the default printer for this PC or laptop. If you want other PCs and laptops to access the local printer go through this topic again for each PC or laptop.
